Affordable, Rigorous CD-Based STAAR Software

Call us at 800-930-8378 for a live demo

STAAR Test Maker software is a CD-based program that allows educators to easily monitor student achievement leading up to the new, more rigorous high-stakes STAAR test.

STAAR Test Maker helps Texas schools create their benchmark and formative assessments to monitor student progress throughout the school year. With STAAR Test Maker, dozens of full-length STAAR tests are now just a few clicks away.
Here are just a few of STAAR Test Maker's features:

  • Easily create and print rigorous, aligned STAAR tests in seconds. How does it work?  
  • Create STAAR-M and Spanish assessments tailored to fit your needs.
  • Includes more than 35,000 STAAR test items aligned to the latest TEKS for all STAAR-tested subjects. Learn more about our item bank.
