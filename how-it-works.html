    Stress-Free Test Creation

    STAAR Test Maker lets you create a test in just a few easy steps:

    1. Select your subject.
    2. Select your grade.
    3. Select the Expectations and Standards you want to assess.
    4. Edit, rearrange, or add your own test items.
    5. Format the test just the way you want it.

    In minutes, you have an aligned, rigorous STAAR Test ready to print. STAAR Test Maker is CD-based software that allows educators to create STAAR assessments in minutes. Learn more about how are test items are created.

    Everything you need to create tests is included on the CD, including our Texas-specific item bank. STAAR Test Maker ships in three editions: Elementary, Middle, and High School EOC.

    STAAR Test Maker can be installed in a shared folder on your school’s server, or on individual computers. STAAR Test Maker is sold as an unlimited site license for your school, so you can install the software wherever you need it.

    To see a live, interactive Web Demonstration of STAAR Test Maker, call us today at 800-930-8378.

    Download a Sample Test created from STAAR Test Maker to see the results.  STAAR Test Maker is designed so that even the most computer-shy teacher can be up and running fast. Now you can improve your instruction while saving your valuable time in the classroom.
