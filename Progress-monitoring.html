Powerful Progress Monitoring

In addition to creating realistic STAAR tests, STAAR Test Maker gives teachers control over their data. The software’s included Progress Monitor allows teachers to create color-coded reports so they can get the information they need, fast.

Score Your Students’ Work

Need to machine-score a test? STAAR Test Maker includes built-in scanner support. Use the supported Apperson Advantage 1200 scanner and get the test results you need in minutes. Learn more about the Advantage 1200 scanner.

Student data, Simplified

STAAR Test Maker’s Progress Reporter allows you to see your formative assessment data in every category you require. Quickly generate reports as you need them, in categories such as:

Student Reports

  • Test Scores - List all of the test scores for an individual student.
  • Standard Progress - Track the student's test-by-test progress by standard.
  • Standard Summary - Break down the student's performance by standard.

Test Reports

  • Statistics - Tabulate important test statistics for an entire class.
  • Student Answers - List the answers and scores for all students in the entire class for the selected test, allowing you to compare students' answers against one another.
  • Item Analysis - List how students performed on each question of a specific test, including common incorrect responses, and list the strand, benchmark, p-value, and discrimination index for each question on the test.
  • Standard Summary - Break down the performance by standard for all students in the class.
  • Standard Detail - See the proficiency levels for students for each standard on a specific test. Students are grouped by proficiency level for each standard.
  • Scores by Demographic Group - Separate the class scores by demographic.

Class Reports

  • Test Scores – List the scores of selected tests for each student in a class.
  • Benchmark Summary - Detail the performance, by standard, of all students.
  • Standard Detail - See the proficiency levels for students for each standard.
  • Standard Progress - Track the class's test-by-test progress by standard.

Comprehensive Reports – Reports for Two or More Classes

  • Test Scores - List the selected test scores for two or more classes side-by-side.
  • Standard Progress - Track test-by-test progress on standards.
  • Standard Summary - Detail performance by standard.
  • Standard Performance by Class - Compare the performance of two or more classes on standards.
  • Standard Detail - Detail the proficiency levels that students in the classes are performing at for each standard.
  • Standard Performance by School (District version only) - Compare the performance of two or more schools.
